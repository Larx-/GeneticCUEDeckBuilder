\section{Checklisten}\label{sec:checklists}
%
Im Folgenden finden Sie einige Checklisten zur Überprüfung Ihrer finalen Arbeit. Die Listen bieten nur einen Anhaltspunkt und erheben keinen Anspruch auf Vollständigkeit. 

%%
%%
\subsection*{Layout}\label{sec:checklists:layout}
%
\begin{itemize}
  \vspace{-3mm}
  \item Das Seitenformat ist DIN A4 
  \item Die Schriftgröße ist beträgt ca. 10-11 pt
  \item Der Zeilenabstand beträgt 1.5
  \item Der linke und rechte Rand beträgt mind. 3 cm
\end{itemize}

%%
%%
\subsection*{Formalien}\label{sec:checklists:formalities}
%
\begin{itemize}
  \vspace{-3mm}
  \item Die eidesstattliche Erklärung ist vorhanden
  \item Die eidesstattliche Erklärung ist unterschrieben
  \item Die englische Zusammenfassung ist vorhanden
  \item Die deutsche Zusammenfassung ist vorhanden
\end{itemize}

%%
%%
\subsection*{Titelblatt}\label{sec:checklists:titlepage}
%
\begin{itemize}
  \vspace{-3mm}
  \item Der Titel der Arbeit ist korrekt geschrieben
  \item Die Art der Arbeit (B.Sc. oder M.Sc.) ist korrekt 
  \item Der Name des Referenten ist korrekt
  \item Der Name des Korreferenten ist korrekt
  \item Das Datum der Abgabe ist korrekt
\end{itemize}

%%
%%
\subsection*{Struktur der Arbeit}\label{sec:checklists:structure}
%
\begin{itemize}
  \vspace{-3mm}
  \item Die Kapitelüberschriften stimmen mit dem Inhaltsverzeichnis überein
  \item Die Kapitelüberschriften enthalten keine Abkürzungen
  \item Jeder Überschrift folgt ein Text mit mind. 2-3 Sätzen
  \item Jedes Kapitel besteht aus 0 oder mind. 2 Abschnitten
  \item Jeder Abschnitt besteht aus 0 oder mind. 2 Unterabschnitten
\end{itemize}

%%
%%
\subsection*{Bilder}\label{sec:checklists:pictures}
%
\begin{itemize}
  \vspace{-3mm}
  \item Alle Bilder sind im Text referenziert
  \item Alle Bilder sind - soweit nötig - mit Quellen versehen
  \item Alle Bilder verfügen über eine aussagekräfte Bildunterschrift (Caption)
  \item Alle Bilder sind im Abbildungsverzeichnis vertreten
  \item Das Abbildungsverzeichnis enthält keine Referenzen
\end{itemize}
